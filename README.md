# play

a WAV decoder and player using the `portaudio` audio interface and standard C++17

## Build

Build by running these from the root of the repository
```sh
cmake -B build -S .
cmake --build build
```

You will find the binary in `./build`

## Usage

```sh
./play WAV_FILE
```
